CHART Me ?

Have you ever asked yourself why data is important? Data is everything. It helps you to make better decisions and allows your business to have a baseline and move forward. Data evaluation is an important part of decision-making. And visual representation can help us to get a better idea. Chart Me is a visualization tool that helps to represent data using different visualization tools to help the human mind comprehend the information easily.

What are your options?

Initially, the following options are available. But don't be dependent. If you are a Developer, feel free to use the source code and develop your version (Please do not use it for any commercial purpose without permission!).

  • Line chart
  • Bar chart
  • Doughnut chart
  • Pie chart
  • Polar area chart
  • Scatter plot

Can you develop your own version?

Of course! It's free for all educational purposes. But please ask before using it for any commercial purpose! You will find the developer information on the linked GitHub account. or use this:

Instructions

View chart: Makes the graph based on inserted points.

Refresh index: Clears your values.

Input points one by one and input both x-axis [ordered] and y-axis together.

In x-axis you can use custom name and in y-axis use values.
